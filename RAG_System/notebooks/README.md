## Архитектура репозитория  

- app.py - код для запуска RAG-системы в веб-интерфейсе  
- uploading_data.ipynb - ноутбук по созданию векторной базы данных и загрузки документов
- new_user_registration.py - скрипт для создания (регистрации) нового пользователя  
- env.yml - conda-окружение (устарело, позже будет новое)  
- chainlit/ (перед запуском переименуйте директорию chainlit -> .chainlit); public/; chainlit_ru.md; .env - файлы для работы и кастомизации системы
